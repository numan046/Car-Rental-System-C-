# 🚗 Car Rental System (C++ Console Project)

## 📌 Description
A simple C++ console-based Car Rental System that manages car bookings, calculates rent based on rental days, and generates a detailed invoice with customer information.

---

## ✨ Features
- Rent multiple cars in one session  
- Enter customer details (name, date)  
- Display available cars with prices  
- Select car by number  
- Calculate rent automatically per day  
- Generate invoice with total bill
